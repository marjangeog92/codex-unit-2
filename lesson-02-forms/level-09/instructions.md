## Level 09 — Basic Validation

Objective: Add `required`, `minlength`, `maxlength`, and set `min`/`max` for range inputs.

Tasks:

- Mark at least one key field as `required`.
- Add `minlength`/`maxlength` to text inputs where appropriate.
- If a range input exists, include `min` and `max` values.

Hints:

- These attributes enable basic browser-side validation without JavaScript.
