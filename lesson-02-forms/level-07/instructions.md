## Level 07 — Labels & Accessibility

Objective: Attach accessible labels to each control. Demonstrate implicit labeling (wrap input in `<label>`) and explicit labeling (`<label for="id">`).

Tasks:

- Ensure every input/select/textarea has an associated label.
- Try both implicit and explicit labeling approaches.

Hints:

- For explicit labels, the `for` attribute value must match the input `id`.
